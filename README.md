# Core77

Core77 is a local, open-source AI app builder. It's fast, private and fully under your control — like Lovable, v0, or Bolt, but running right on your machine.

![Image](https://github.com/user-attachments/assets/f6c83dfc-6ffd-4d32-93dd-4b9c46d17790)

More info at: [http://core77.com/](http://core77.com/)

## 🚀 Features

- ⚡️ **Local**: Fast, private and no lock-in.
- 🛠 **Bring your own keys**: Use your own AI API keys with no vendor lock-in.
- 🖥️ **Cross-platform**: Easy to run on Mac or Windows.
